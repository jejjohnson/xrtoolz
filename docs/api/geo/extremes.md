# Extremes

Block-extrema (block maxima / minima for a generalized extreme-value
workflow) and peaks-over-threshold (POT) diagnostics: threshold selection,
exceedances, and Poisson-process counts and statistics.

These pure functions back the operators above; each takes `xr.DataArray`/`xr.Dataset` and a `dim:` argument.
